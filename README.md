# Template for CV-Portfolio webpage

[Show webpage here](https://gaetantremois.fr)

## :point_right: You can use this template for yours own sites :exclamation:

### How to ?

```
mkdir <directory>
git clone https://github.com/gtcore902/webapp-videosurveillance-raspberry-python.git
```

You need to update these variables in sendFormV.php in your code editor to use form section:
* $dest = "your email"
* $from = "email from to send form" (you can choose any)
* $dbname = "your database name"
* $login = "your login"
* $pass = "your password"
* Finally you need to replace **<formulaire_cv_new>** with your database table name
